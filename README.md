# iReap Asset Mockup

Clickable static prototype for the Asset Label Printing workflow.

## Run Locally

Open `index.html` in a browser from VS Code, or use any simple static file server.

## Included Flows

- Landing page with Web Mockup and Mobile Mockup entry points
- Web Asset submenu with Asset List and Print Labels, flexible Print Templates, Print Options modal, and Label Preview
- Mobile Home, Setup, flexible Print Templates, Assets List, Asset Detail, Print Options bottom sheet, and Label Preview

All data is dummy static data. PDF, print, share, and unrelated menu actions show mockup-only feedback.
